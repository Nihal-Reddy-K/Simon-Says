Simon Says Game

It is a fun and interactive wed based app built using HTML,CSS and JavaScript.
Test your memory by repeating the sequence of colors! One wrong move and the game's over.

Features:
-Level-based progression

-Button flash animations

-Game starts using keyboard input

-Game-over feedback with score

-Simple and beginner-friendly

Link: 
https://nihal-reddy-k.github.io/Simon-Says/

How to Play:
1.Press any key to start the game

2.Watch the color sequence carefully

3.Click the buttons in the same order

4.Each level adds a new color to the sequence

5.If you click the wrong button, the game ends
